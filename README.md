# Student Management System

A full-stack web application for managing students and their exam records, built with Angular frontend and Django REST API backend.

## Features

### Student Management
- Add, edit, delete students
- View student list with details
- Student ID validation
- Responsive form design

### Exam Management
- Record exam marks per subject
- Automatic grade calculation (A, A-, B+, B, B-, C+, C, C-, D, E)
- Filter exams by student, subject, term
- Percentage calculation
- Exam date tracking

## Tech Stack

**Frontend:**
- Angular 16+
- TypeScript
- Bootstrap 5
- RxJS

**Backend:**
- Django 4+
- Django REST Framework
- SQLite (development)
- PostgreSQL ready
